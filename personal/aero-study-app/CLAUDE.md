# AeroStudy

Interactive aerospace engineering study app for learning key principles through simulation.

## Goal
Help an aerospace engineering student understand core topics via interactive, parameter-driven simulations.

## Tech Stack
- Python + Streamlit (UI and interactivity)
- NumPy / SciPy (numerical methods)
- Plotly (interactive charts)
- Matplotlib (static diagrams)

## Run
```
pip install -r requirements.txt
streamlit run app.py
```

## Modules
- Aerodynamics — NACA airfoils, lift/drag
- Compressible Flow — isentropic relations, normal shocks
- Propulsion — Tsiolkovsky rocket equation, nozzle flow
- Orbital Mechanics — two-body orbit, Hohmann transfer
- Flight Mechanics — trim, V-n diagram
- Structures — beam bending, pressure vessel
